# カレーのレシピ
## 材料
豚肉 - 250g
じゃがいも - 2個
にんじん - 1本
たまねぎ - 2個
水 - 800cc
サラダ油 - 大さじ1
カレールウ - 半分
隠し味 - はちみつ,　MonsterEnergy
